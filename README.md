# Falling Fruits 

This project build for understanding Unity Editor and creating something in 2 dimensional game type. 

# Getting Started 

- Download the Falling Fruits.apk 
- Open the Game App. 
- And press the `Click to play`. 
- Then you can enjoy the game. 
  ```bash 
    Only for the Android Users. 
  ``` 

# Usage/Examples 

- You must click the healthy fruits so you can collect the points and become healthy. Otherwise you become poisoned.
